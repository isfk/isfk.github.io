## 直接开始吧

```bash
# 安装 ionic cli 工具
npm install -g @ionic/cli@testing

# 初始化一个项目
ionic start app tabs --type vue --tag vue-beta

# 浏览器预览页面
ionic serve

# 生成静态文件到 dist 目录
npm run build

# 生成原生代码 android/ios
ionic capacitor add android

# 使用 android studio 打开项目
```
